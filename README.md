# 🛍️ Avanti - Loja Online (Landing Page)

Este projeto é uma landing page de uma loja online fictícia chamada **Avanti**, focada em produtos modernos, com destaque para categorias, lançamentos e banners promocionais.

## 🚀 Funcionalidades

- Menu interativo com múltiplos níveis de dropdown.
- Swipers (carrosséis) responsivos para banners e produtos.
- Campo de busca com feedback do valor pesquisado (desktop e mobile).
- Seções informativas, newsletter e footer com links úteis.
- Design moderno, responsivo e acessível.

## 🖼️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- [**Swiper.js**](https://swiperjs.com/) – carrossel de banners e lançamentos
- **Google Fonts** – Nunito Sans
- **Font Awesome** e **Material Icons**

---

## 📱 Responsividade

- Layout adaptado para **desktop** e **mobile**
- Campos de busca e banners ajustados conforme o dispositivo
- Menus principais ocultos no mobile para melhor usabilidade

---

## 🛠️ Como Rodar o Projeto

1. Clone o repositório ou baixe o arquivo `.zip`:
   ```bash
   git clone https://github.com/seu-usuario/avanti-landingpage.git
   ```

2. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge...).

> ✅ **Não é necessário instalar dependências ou rodar comandos.**  
> Todo o projeto é feito com arquivos estáticos.
